波形繪製

添加波形資訊

  1. 輸入波形名稱。
  2. 選擇波形顏色。
  3. 按下添加按鈕。
  4. 剛剛添加的波形名稱將顯示在波形清單中。

繪製

當串口收到符合波形繪製格式的字串時，會自動繪製到波形介面。

波形格式

波形名稱 等號 值 結束符
X1 = 200 \r\n

使用"\r\n"作為一條繪製的結束
多條波形可使用英文 "," 分割.
這是演示。 波形名稱 區分大小寫 

"X1=100,"
"X1=100\r\n"

"X1=100,X2=200,"     printf("X1=100,X2=230,");
"X1=100,X2=200\r\n"  printf("X1=100,X2=230\r\n");

注意使用print（）函數時使用 “\r\n” 來表示回車換行符。 軟體發送區可以直接按回車輸入回車換行符，不可見但游標會移動到下一行，可以點擊十六進位發送會發現字元會有16個十進位數0D 0A 。

座標軸設置

操作說明

  1. Y軸坐標自動調整。
    開：根據輸入數據自動調整坐標範圍。
    關：需要自己輸入坐標最大值和最小值。
  2. 自動暫停： 波形長度超過一屏顯示時，按下鼠標左鍵並向右滑動，會自動啟動波形暫停功能，方便對波形進行觀測。

縮放

  1. Ctrl + 滾輪 -> 波形縮放
  2. X + 滾輪 -> 單獨對X軸坐標範圍進行縮放。
  3. Y + 滾輪 -> 單獨對Y軸坐標範圍進行縮放。
